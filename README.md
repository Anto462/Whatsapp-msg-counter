# English
----------------------------------------------------------------------
This program is designed to be simple and intuitive. On several occasions, small businesses have asked me how they could keep track of the number of messages exchanged in customer chats, but they are often scared to use third-party tools.

With the goal of supporting anyone facing this situation, I decided to release this base code. It includes everything necessary to accomplish the main objective, along with an intuitive UI to make it accessible even for users without advanced knowledge of software tools or computer management.

The intention of this program is to be used as-is, or as a starting point that developers can modify to better fit their own needs. With this base, it is possible to build many features and extract a significant amount of information from an exported WhatsApp chat. I chose to publish only the core code to keep the project simple and to encourage developers to adapt it according to their requirements.

If you use this project, I would appreciate a simple written credit in the comments. nothing more is required. Good luck!
----------------------------------------------------------------------
Code Explanation

- First, it is important to export the chat you want to analyze from your mobile device.
<img width="336" height="196" alt="Export" src="https://github.com/user-attachments/assets/f14c4e4a-7521-49a6-a4d5-80273968833b" />

- This will generate a .zip file. Once extracted, it will contain a .txt file with the full chat data. After that, you can start the program.
